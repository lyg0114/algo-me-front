## 💡로컬실행
```shell
# api 실행
git clone https://github.com/lyg0114/algo-mi-backend.git
cd algo-mi-backend
./gradlew clean build
java -jar ./build/libs/app.jar --spring.profiles.active=local

# 화면 실행
git clone https://github.com/lyg0114/algo-mi-frontend.git
cd algo-mi-frontend
yarn install
yarn start 
```
